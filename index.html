Soft Keyboard-Based Text Entry System Using Eye Blinking


• Created a user-friendly application (with three members in a team) that helps to operate smartphones for handicapped people using eye blinking.
• The application can be considered an alternative solution like augmentative and alternative communication (AAC) for either abled-bodied or differently abled people suffering from multiple disabilities except blindness.
• This model works with the help of an Eye Blinking Sensor, an Uno Arduino, a Bluetooth Module, and a Soft Keyboard-Based Text Entry System Using Eye Blinking Applications.
• Benefits: Low cost and affordability compared to existing technologies, easy interaction with and detection of input signals & enabling communication with hardware and software through eye blinking.
• Target users: People with disabilities like stroke, spinal cord injuries, speaking impairment, and neuromotor diseases
